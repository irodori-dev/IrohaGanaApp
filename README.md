# IrohaGanaApp

🍎 iPhone / iPad<br>
<br>
いろは仮名 - 日本語ルビ出力アプリケーション

## Features
 
日本語文字列をひらがなに変換 ♪<br>

```bash
(example)
"徒然なるままに、日暮らし" -> "つれづれなる ままに ひぐらし"
```
